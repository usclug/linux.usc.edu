Using the Command Line: The Basics, Tips and Tricks

Time: Thursday March 13, 2014 6:30-8pm

Location: THH102

Facebook Event

Interactive workshop where you learn how to use the command line (terminal) in Linux and Unix (like Mac OS X) systems. You'll learn about the basics that every developer and programmer needs, and learn the commands you need to be productive in Linux and Unix environments.

USCLUG will be hosting a workshop this week about the command line interface (terminal) this Thursday! Intended for beginning programmers and non-programmers, this workshop will give you the foundation necessary to start being a better developer and using Linux. We'll start with the basics, and then get to a point where we can start creating basic websites on servers.

This workshop will serve as the foundation for our later workshops, so be sure to come out and learn! :D

If you want to actively participate in the workshop, we would like you to be actually working in Linux. You can do this by using a Virtual Machine and use Linux if you are not already using it.

  1. Download the Linux Virtual Machine at this link.
  2. Install VirtualBox and import the VM (username: usclug / pw: tux).
