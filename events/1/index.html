"Building a MEAN Web Stack": AWS Cloud Hosting Workshop

Time: Thursday October 3, 2013 7-9pm

Location: ZHS159

Facebook Event

Interactive workshop where you learn about hosting javascript apps on the cloud (Amazon EC2).

Have you ever wanted to build a a web app on the cloud, but was not sure where to start? Are you a web development newbie? This Thursday, USCLUG will be hosting a workshop to teach you how to build a 'MEAN' javascript application on Amazon Web Services!

Ankit Tyagi (Software Engineer at Dun & Bradstreet Credibility Corp) and Sheldon Kwok (Software Engineer at Second Spectrum) will help you setup and build your own 'MEAN' (MongoDB, Express, AngularJS, Node.js) web app on a free Amazon EC2 Micro Instance.

Be sure to bring your laptop and charger! ...and no experience in web development required! Show up and learn. :D
