GIT: Git Isn't Tough

A Workshop on Version Control and Social Coding Using Git


Interactive workshop where you learn about the popular software Git and the workflow behind it, and how to use version control in a professional and collaborative setting.

Location: SLH 200
Date: Thursday, Oct 24th
Time: 7pm to 9pm
What to bring: your laptop (and charger)


Event Details

Have you ever heard about Git and are confused by the magic of it? Or are you an experienced Git user looking for some additional tips and tricks under your belt? This Thursday, USCLUG will be hosting a workshop to teach you how to use Git and show you that Git isn't tough. We'll teach you how Git works, discuss some best practices and show how you can collaboratively work with other programmers using version control. We'll also reveal some of the magic behind Git and how you can contribute using GitHub (and beef up your resume while you're at it)!

We'll have lots of GitHub swag to giveaway, so be sure to come promptly at 7pm!

Building a MEAN Web Stack

Location: ZHS 159
Date: 10/03 Thursday
Time: 7pm to 9pm
What to bring: your laptop (and charger)


Event Details

Have you ever wanted to build a MEAN web stack but was never sure where to start? Well, you're in luck! This Thursday, the USC LUG will be hosting an event to teach you just how to do that. Ankit Tyagi (Software Engineer at Dun & Bradstreet Credibility Corp) and Sheldon Kwok (Software Engineer at Second Spectrum) will be having a hands-on class to setup and build your own MEAN (Mongo, Express, AngularJS, Node) web app on a FREE Amazon EC2 Micro Instance. Pizza and drinks will be provided!


Technologies Used

MongoDB

A NoSQL document database that provides high performance, high availability, and easy scalability.

NodeJS

Node.js is a library written for Google Chrome's V8 JavaScript engine used for server-side programming with JavaScript.

ExpressJS

Express is a minimal and flexible node.js web application framework, providing a robust set of features for building single and multi-page, and hybrid web applications.

AngularJS

AngularJS is a client-side web framework used to create dynamic apps. It allows you to use HMTL as a templating language and eliminates much of the code you currently write through data binding and dependency injection.
