Using the Command Line: The Basics, Tips and Tricks

Time: Thursday March 13, 2014 6:30-8pm

Location: THH102

Facebook Event

Interactive workshop where you learn how to use the command line (terminal) in Linux and Unix (like Mac OS X) systems. You'll learn about the basics that every developer and programmer needs, and learn the commands you need to be productive in Linux and Unix environments.

USCLUG goes to SoCal Linux Expo

Time: Friday February 21 - Sunday February 23, 2014

Location: Hilton LAX

Facebook Event

USCLUG (the people who brought you GIT and AWS cloud hosting workshops) will be heading to SCALE, the first and biggest open-source conference in Southern California. With lots of technical workshops, and a career expo to boot, there's lots of options for people interested and involved in software development and open-source. Sessions include workshops on OpenStack, PostgreSQL, Chef, Puppet, Cloud Technologies, DevOps, System Administration, Hardware Hacking, Security, etc.
