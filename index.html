GIT: Git Isn't Tough

A Workshop on Version Control and Social Coding Using Git


Interactive workshop where you learn about the popular software Git and the workflow behind it, and how to use version control in a professional and collaborative setting.

Location: SLH 200
Date: Thursday, Oct 24th
Time: 7pm to 9pm
What to bring: your laptop (and charger)


Event Details

Have you ever heard about Git and are confused by the magic of it? Or are you an experienced Git user looking for some additional tips and tricks under your belt? This Thursday, USCLUG will be hosting a workshop to teach you how to use Git and show you that Git isn't tough. We'll teach you how Git works, discuss some best practices and show how you can collaboratively work with other programmers using version control. We'll also reveal some of the magic behind Git and how you can contribute using GitHub (and beef up your resume while you're at it)!

We'll have lots of GitHub swag to giveaway, so be sure to come promptly at 7pm!
